            🆘 Help & Support

            Aam taur par aane wali problems ke solutions niche diye gaye hain. Kisi bhi error par click karein:

            ❌ Login Failed / Wrong ID Password?▼
            Kripya dhyan dein ki aapne sahi Role (Owner, Milk Man, Customer) select kiya hai. Apna ID aur Password check karein. Agar aap password bhool gaye hain, toh Owner se sampark karein.
            🌐 Connection Failed / Network Error?▼
            Apna internet connection check karein. Agar aap local computer par hain, toh ensure karein ki Backend server (Python) chalu hai aur screen par "Backend Chal Raha Hai..." likha aa raha hai.
            📅 Kal ka doodh band karna/badalna hai?▼
            Apne Customer dashboard mein jayein, "My Order" (🛒) tab select karein, wahan "One-Time Change" mein agle din ki date select karein aur quantity ko update/0 karke save kar dein.
            🔄 Data sync ya update nahi ho raha?▼
            Page ko refresh karein, ya logout karke dobara login karein. Agar phir bhi problem aaye toh ensure karein background data properly sync ho gaya hai.
            🔑 Password Reset/Change Kaise Karein?▼
            Owner apne dashboard se "Customers" ya "Milk Men" list mein jakar "Pass" button daba kar kisi ka bhi password reset kar sakte hain. Khud ka password badalne ke liye upar "Profile" menu me jayein.
            📦 Naya Product ya Rate kaise add karein?▼
            "Manage Products" (📦) menu mein jayein. Wahan aap doodh ke alawa naye items (jaise Ghee, Paneer) aur unka naya rate add ya update kar sakte hain.
            💸 Customer Online Payment kaise karein?▼
            Customer apne portal me "Payment" (💸) tab me jayen, "Online" select karein, Owner ka QR code scan karke payment karein aur screenshot upload karke submit kar dein.
            🔢 Delivery Sequence (Kram) kaise badlein?▼
            Owner ya Milk Man "Daily Feed" tab me "Edit Seq" button par click karke subah ya shaam ke hisaab se customers ko nayi sequence (1, 2, 3...) de sakte hain.
            🧾 Bina khate wale customer ka bill kaise banayein?▼
            "General Bill" (🧾) section ka use karein. Naye customer ka naam, mobile no. aur items daal kar turant invoice / receipt print kar sakte hain.
            💳 Apna QR Code ya Profile Photo kaise badlein?▼
            Upar right corner mein User Profile (👤) par click karein. "Edit Profile" daba kar aap apni photo, details, aur Payment QR code update karke save kar sakte hain.
